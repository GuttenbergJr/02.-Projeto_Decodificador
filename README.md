<h1>Codificador Web</h1>

# Sobre o projeto
- O projeto consiste em um site que pode codificar e decodificar mensagens nas seguintes linguagens, Base64 e Cifra de Cesar. 

# Teconolgias utilizadas 
<li>Html</li>
<li>Css</li>
<li>Javascript</li>
